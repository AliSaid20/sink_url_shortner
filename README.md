A powerful and minimal **URL shortening web application** built with **React + Vite**, **FastAPI**, and **MongoDB Atlas**.  
It allows users to shorten long URLs, customize aliases, set expiration dates, and generate customizable **QR codes**.

 **Demo Video:** [Watch on LinkedIn](https://www.linkedin.com/posts/muhammed-saidali-s-596062346_sink-urlshortener-responsivedesign-activity-7304108645754028032-9Y69?utm_source=share&utm_medium=member_android&rcm=ACoAAFaBVsIBjcnOuQnFYAcMNSKJozu-ROtcVmI)
